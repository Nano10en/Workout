# Workout (Vue 3)

A small Vue 3 SPA/website: Home and Pricing pages via Vue Router, component-based structure (courses/benefits/video), responsive layout, SCSS styling, and Webpack build.

## Features
- Vue 3 + components
- Navigation (Vue Router): Home / Pricing
- Responsive UI
- SCSS styles
- Webpack build
