# Workout (Vue 3 + Webpack)

Small Vue 3 SPA with Vue Router (Home/Pricing), reusable components, responsive layout, and SCSS styling.

## Features
- Vue 3 components
- Vue Router: Home / Pricing
- Responsive UI
- SCSS
- Webpack build
